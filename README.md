# Investment Platform

A modern investment platform built with Next.js, TypeScript, and Tailwind CSS.

## Features

- 🎨 Beautiful UI with braxley-inspired color palette (Metallic Gold theme)
- 🌓 Dark mode support
- ⚡ Built with Next.js 15
- 📱 Responsive design
- 🎯 TypeScript for type safety

## Getting Started

First, install the dependencies:

```bash
npm install
```

Then, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Color Palette

This project uses the braxley color palette:
- **Primary**: Metallic Gold (#C5A059)
- **Background**: White (light) / Near-black (dark)
- **Foreground**: Dark charcoal (light) / Light grey (dark)

## Tech Stack

- Next.js 15
- React 19
- TypeScript
- Tailwind CSS
- Framer Motion
- Radix UI
