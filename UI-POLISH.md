# AlphaTrader UI/UX Polish Roadmap

> **Problem:** The site feels static, corporate, and lifeless. It doesn't convey the energy, excitement, and trust that prop traders expect.
>
> **Goal:** Transform it into a premium, high-energy trading platform that screams "funded trader success."

---

## 🎯 Priority 1: Visual Identity Overhaul

### 1.1 Color System Revolution
- [ ] Replace muted greens with **electric emerald** (#00FF88) for profit/success
- [ ] Add **neon cyan** accent (#00D4FF) for highlights
- [ ] Implement **deep navy** (#0A0F1C) instead of pure black backgrounds
- [ ] Create gradient system: emerald → cyan for CTAs
- [ ] Add subtle **purple glow** (#8B5CF6) for premium elements
- [ ] Design "profit green" and "loss red" that pop

### 1.2 Typography Upgrade
- [ ] Replace Inter with **Space Grotesk** or **Satoshi** for headings
- [ ] Use **JetBrains Mono** or **IBM Plex Mono** for numbers/stats
- [ ] Implement variable font weights for better hierarchy
- [ ] Add letter-spacing to uppercase labels
- [ ] Create display sizes: 72px+ hero headlines
- [ ] Design "trading terminal" aesthetic for data

### 1.3 Depth & Dimension
- [ ] Add glassmorphism cards with backdrop blur
- [ ] Implement layered shadows (not flat)
- [ ] Create subtle noise/grain texture overlay
- [ ] Add gradient mesh backgrounds
- [ ] Design "glow" effects behind key elements
- [ ] Use border gradients instead of solid borders
